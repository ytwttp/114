### 1️⃣ Gauss Elimination with Pivoting


อัลกอริทึมสำหรับแก้ระบบสมการเชิงเส้นโดยใช้วิธี **Gaussian Elimination** ร่วมกับ **Partial Pivoting**
เพื่อเพิ่มความเสถียรทางตัวเลข (Numerical Stability) และลดความผิดพลาดที่เกิดจากการหารด้วยค่าที่เข้าใกล้ศูนย์


**แนวคิดหลัก**


* แปลงเมทริกซ์ให้อยู่ในรูป Upper Triangular
  
* สลับแถว (Row swapping) เพื่อเลือก pivot ที่เหมาะสม

* แก้สมการด้วยวิธี Back Substitution

### 2️⃣ Gauss–Jordan Elimination


วิธีการแก้ระบบสมการเชิงเส้นโดยขยายจาก Gaussian Elimination

ซึ่งสามารถแปลงเมทริกซ์ให้อยู่ในรูป **Reduced Row Echelon Form (RREF)**


**จุดเด่น**


* สามารถหาคำตอบของระบบสมการได้โดยตรง
  
* ใช้เป็นพื้นฐานในการหา Inverse Matrix
  
* เหมาะสำหรับระบบสมการขนาดเล็กถึงกลาง

  

---


### 3️⃣ LU Factorization


การแยกเมทริกซ์ ( A ) ออกเป็นผลคูณของเมทริกซ์สามเหลี่ยมล่าง ( L ) และสามเหลี่ยมบน ( U )


[ A = LU ]


**ประโยชน์**


* ลดเวลาในการคำนวณเมื่อแก้ระบบสมการหลายครั้ง
  
* ใช้งานร่วมกับ Forward และ Back Substitution
  
* เป็นพื้นฐานของวิธีเชิงตัวเลขขั้นสูง

  

---


### 4️⃣ Inverse Matrix


การคำนวณ **เมทริกซ์ผกผัน (Inverse Matrix)** โดยอาศัยวิธี Gauss–Jordan Elimination


**เงื่อนไข**


* เมทริกซ์ต้องเป็น Square Matrix
  
* Determinant ต้องไม่เท่ากับศูนย์


**การประยุกต์**


* ใช้แก้ระบบสมการในรูป ( AX = B )
  
* ใช้ในงานทางวิศวกรรม วิทยาการข้อมูล และฟิสิกส์
